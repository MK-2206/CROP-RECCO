# 🌱 CROP-RECCO: Smart Crop Recommendation System

**CROP-RECCO** is a machine learning-powered crop recommendation tool that suggests the best crops to grow based on real-time environmental and soil conditions. Trained on real agricultural data, it helps improve yield and farming efficiency by identifying the most suitable crops for given conditions.

## 📊 Features

- 🌾 Predicts optimal crops using N, P, K, temperature, humidity, pH, and rainfall
- 🤖 Uses classification models like Random Forest and Support Vector Machine
- 📈 Data-driven predictions for improved agricultural decision-making
- 🧪 Real-time input of soil/environmental data
- 🌍 Optional Streamlit UI for easy interaction

## 🧬 Input Parameters

- Nitrogen (N)  
- Phosphorous (P)  
- Potassium (K)  
- Temperature (°C)  
- Humidity (%)  
- pH  
- Rainfall (mm)

## 🛠️ Tech Stack

- Python  
- Scikit-learn  
- Pandas & NumPy  
- Matplotlib / Seaborn (for visualization)  
- Streamlit (optional frontend)
